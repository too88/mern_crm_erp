
### How to run

1. move to backend direction from #root

   > cd backend

1. install dependencies

   > npm install

1. run seeder (generate data)

   > npm run setup:dev 

1. run API

   > npm start dev # API running at 8888

### How to deploy


1. run seeder (generate data)

   > npm run setup:dev 

1. run API

   > npm start dev # API running at 8888
